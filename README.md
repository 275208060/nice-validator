```
$ bower install nice-validator
```

### Build
First, you need [node.js](http://nodejs.org/) v0.10+ installed.
Then first run the following command to install dependencies:
```
$ npm install -g gulp
$ npm install
```
Run the unit test and build:
```
$ gulp
```
